Tailwind CSS menyediakan banyak kelas yang dapat digunakan untuk membangun tampilan web dengan cepat dan efisien. Berikut adalah beberapa kelas penting yang sering digunakan dalam Tailwind CSS:

1. **Pengaturan Margin dan Padding**:

   - `m-{nilai}`: Mengatur margin (misalnya, `m-4` akan memberikan margin 1rem).
   - `p-{nilai}`: Mengatur padding (misalnya, `p-4` akan memberikan padding 1rem).
  ```html
   <div class="m-4 p-4">Margin dan Padding</div>
   ```

2. **Warna Latar Belakang dan Teks**:
   - `bg-{warna}`: Mengatur warna latar belakang (misalnya, `bg-blue-500` akan memberikan latar belakang biru).
   - `text-{warna}`: Mengatur warna teks (misalnya, `text-red-600` akan memberikan warna teks merah).
   ```html
   <div class="bg-blue-500 text-white">Latar Belakang Biru dan Teks Putih</div>
   ```


3. **Ukuran Font dan Berat Teks**:
   - `text-{ukuran}`: Mengatur ukuran font teks (misalnya, `text-xl` akan memberikan ukuran font besar).
   - `font-{berat}`: Mengatur berat font teks (misalnya, `font-semibold` akan memberikan teks setengah tebal).
   ```html
   <p class="text-xl font-semibold">Teks Besar dan Setengah Tebal</p>
   ```

4. **Flexbox dan Grid**:
   - `flex`: Mengatur elemen sebagai kontainer flexbox.
   - `grid`: Mengatur elemen sebagai kontainer grid.
   - `justify-{nilai}`: Mengatur perataan horizontal dalam flexbox atau grid (misalnya, `justify-center` akan meratakan ke tengah).
   - `items-{nilai}`: Mengatur perataan vertikal dalam flexbox atau grid (misalnya, `items-center` akan meratakan ke tengah).
  ```html
   <div class="flex justify-center items-center">Flexbox Centered</div>
   ```

5. **Border and Shadow**:
   - `border`: Menambahkan garis tepi pada elemen.
   - `shadow-{nilai}`: Menambahkan bayangan pada elemen (misalnya, `shadow-md` akan memberikan bayangan medium).
  ```html
   <div class="border shadow-md">Border and Shadow</div>
   ```

6. **Responsif**:
   - `sm`, `md`, `lg`, `xl`: Menggunakan kelas responsif untuk mengubah perilaku elemen pada berbagai ukuran layar (misalnya, `sm:hidden` akan menyembunyikan elemen pada layar kecil).

  ```html
   <div class="hidden sm:block">Hanya Ditampilkan di Layar Lebar (>= sm)</div>
   ```
7. **Rounded Corners**:
   - `rounded`: Memberikan sudut elemen yang melengkung.
   - `rounded-{nilai}`: Mengatur tingkat melengkung elemen (misalnya, `rounded-full` akan memberikan elemen bentuk bulat).
```html
 <div class="rounded-full">Elemen dengan Sudut Bulat</div>
```
8. **Display**:
   - `hidden`: Menyembunyikan elemen.
   - `block`, `inline-block`, `inline`, `flex`, `grid`, dll.: Mengatur tampilan elemen.

9. **Overflow**:
   - `overflow-{nilai}`: Mengatur perilaku elemen yang melebihi batas (misalnya, `overflow-scroll` akan menampilkan bilah geser jika kontennya melebihi).

10. **Transisi dan Animasi**:
    - `transition-{properti}`: Mengatur animasi transisi (misalnya, `transition-opacity` akan mengatur transisi perubahan opacity).

11. **Hover dan Focus State**:
    - `hover:{kelas}`: Mengatur tampilan elemen saat dihover.
    - `focus:{kelas}`: Mengatur tampilan elemen saat fokus.

12. **Pseudo-class**:
    - `first-child`, `last-child`, dll.: Menggunakan pseudo-class untuk memilih elemen berdasarkan posisinya dalam struktur DOM.

13. **Responsif**:
    - `md:{kelas}`, `lg:{kelas}`, dll.: Menggunakan kelas responsif untuk mengubah perilaku elemen pada berbagai ukuran layar.

14. **Variabel Kustom**:
    - Anda juga dapat membuat kelas kustom menggunakan ekstensi Tailwind atau konfigurasi kustom.
