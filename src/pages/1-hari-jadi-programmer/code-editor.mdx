**Mengenal Text Editor - Menggunakan Visual Studio Code**

Dalam Bab 2 ini, kita akan memperkenalkan Anda pada salah satu text editor yang paling populer untuk pengembangan web, yaitu **Visual Studio Code (VS Code)**. Text editor adalah alat yang sangat penting dalam dunia pemrograman karena digunakan untuk menulis, mengedit, dan mengelola kode Anda.

### Apa itu Visual Studio Code?

Visual Studio Code (VS Code) adalah text editor kode sumber terbuka yang dikembangkan oleh Microsoft. Ini adalah alat yang sangat kuat dan serbaguna yang cocok untuk pengembangan berbagai jenis aplikasi, termasuk pengembangan web. Beberapa fitur menonjol dari VS Code meliputi:

1. **Lintas Platform:** VS Code tersedia untuk Windows, macOS, dan Linux, sehingga Anda dapat menggunakannya di berbagai sistem operasi.

2. **Ekstensibilitas:** Anda dapat menginstal ekstensi tambahan untuk memperluas fungsionalitas VS Code sesuai dengan kebutuhan Anda. Ada ekstensi yang dapat membantu Anda dalam pengembangan HTML dan CSS.

3. **Integrasi Git:** VS Code memiliki integrasi Git yang kuat, memungkinkan Anda untuk melacak perubahan kode Anda dengan mudah.

4. **Autocomplete:** Fitur autocomplete akan membantu Anda menulis kode lebih cepat dengan memberikan saran saat Anda mengetik.

5. **Debugger Terintegrasi:** Anda dapat meng-debug kode Anda langsung dari VS Code.

### Cara Instalasi Visual Studio Code

Untuk memulai menggunakan Visual Studio Code, Anda perlu menginstalnya terlebih dahulu. Berikut adalah langkah-langkah instalasi pada beberapa platform utama:

#### **1. Instalasi di Windows:**

   - Unduh file instalasi VS Code dari situs resmi (https://code.visualstudio.com/download).
   - Buka file installer yang telah diunduh.
   - Ikuti instruksi instalasi yang muncul di layar.
   - Setelah selesai, Anda bisa membuka VS Code dengan mencarinya di menu Start atau melalui desktop.

#### **2. Instalasi di macOS:**

   - Unduh file .dmg instalasi VS Code dari situs resmi (https://code.visualstudio.com/download).
   - Buka file .dmg dan seret ikon VS Code ke folder Aplikasi.
   - Buka VS Code dari folder Aplikasi atau melalui Launchpad.

#### **3. Instalasi di Linux (Ubuntu):**

   - Buka Terminal.
   - Jalankan perintah berikut untuk mengunduh dan menginstal VS Code:
     ```
     sudo apt update
     sudo apt install code
     ```
   - Setelah selesai, Anda dapat menjalankan VS Code dengan perintah `code` dari Terminal atau mencarinya di aplikasi.

Sekarang, Anda telah berhasil menginstal Visual Studio Code di sistem Anda. Dalam bab-bab berikutnya, kita akan belajar cara menggunakannya untuk menulis dan mengedit kode HTML serta CSS. Tetap semangat dalam perjalanan Anda menjadi seorang programmer!
