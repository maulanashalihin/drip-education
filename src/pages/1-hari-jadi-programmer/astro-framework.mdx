**Mengenal Astro - Framework Modern untuk Pengembangan Web**

Dalam Bab ini, kita akan memperkenalkan **Astro**, sebuah framework modern untuk pengembangan web yang akan mempermudah Anda dalam membangun situs web yang cepat, efisien, dan mudah di-maintain. Astro memungkinkan Anda untuk menggunakan komponen Svelte, React, Vue, atau bahkan HTML murni dalam proyek Anda.

### Apa itu Astro?

Astro adalah framework pengembangan web yang memberikan pendekatan baru dalam membangun situs web. Ini memiliki beberapa fitur unik:

1. **Bekerja dengan Kinerja Tinggi:** Astro menghasilkan halaman statis dengan performa tinggi secara otomatis, membuat waktu muat yang cepat dan situs yang responsif.

2. **Pemilihan Komponen:** Anda dapat menggunakan komponen Svelte, React, Vue, atau bahkan HTML biasa dalam proyek Astro, memberi Anda fleksibilitas dalam memilih alat yang sesuai dengan kebutuhan Anda.

3. **Pembuatan Situs yang Ramah Jaringan:** Astro memungkinkan Anda untuk menghindari over-fetching dan hanya mengunduh kode JavaScript yang benar-benar diperlukan oleh halaman, membuat situs yang ramah jaringan.

### Cara Instalasi

sebelum menginstal Astro, penting untuk memastikan bahwa Node.js telah diinstal di komputer Anda. Di bawah ini adalah langkah-langkah instalasi Astro dengan menyertakan prasyaratnya:

#### **1. Pastikan Anda Memiliki Node.js Terinstal:**

Sebelum menginstal Astro, pastikan Anda telah menginstal Node.js. Anda dapat mengunduh Node.js dari situs resmi di [https://nodejs.org/](https://nodejs.org/). Pilih versi LTS (Long Term Support) untuk stabilitas yang lebih baik.

#### **2. Buat Proyek Baru:**

Buka terminal Anda dan buat direktori untuk proyek Astro Anda. Masuk ke direktori tersebut.

```bash
mkdir folder-project
cd folder-project
```

#### **3. Inisialisasi Proyek Astro:**

Jalankan perintah berikut untuk menginisialisasi proyek Astro baru.

```bash
npx create-astro learn-html
```

Ikuti langkah-langkah yang muncul di layar untuk menentukan pilihan Anda.

#### **4. Memulai Pengembangan:**

Setelah inisialisasi selesai, Anda dapat memulai pengembangan situs Anda dengan menjalankan server pengembangan Astro:

```bash
npm run dev
```

Astro akan memberikan URL lokal (biasanya http://localhost:3000/) di mana Anda dapat melihat situs web Anda saat ini. Selanjutnya, Anda dapat mulai menulis kode HTML, CSS, dan JavaScript dalam proyek Astro Anda seperti yang Anda inginkan.

Dengan Astro, Anda dapat merancang situs web yang cepat dan efisien dengan cepat. Anda sekarang telah menambahkan prasyarat yang diperlukan untuk instalasi Astro. Selamat bersenang-senang dalam menjelajahi Astro dan mengembangkan situs web yang luar biasa!
