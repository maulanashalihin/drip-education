**Mengenal HTML - Bahasa Pemrograman Web Pertama Anda**

Kami akan memulai perjalanan Anda menjadi seorang programmer dengan mengenalkan bahasa pemrograman web yang paling mendasar dan penting, yaitu **HTML (HyperText Markup Language)**. HTML adalah fondasi dari hampir semua situs web yang Anda lihat di internet. Ini adalah bahasa yang digunakan untuk membuat struktur dasar dari sebuah halaman web.

### Mengapa HTML Penting?

HTML adalah bahasa yang memungkinkan kita untuk mengatur konten di halaman web. Anda dapat memikirkannya seperti kerangka atau struktur dasar dari sebuah bangunan. Ini mencakup elemen-elemen seperti teks, gambar, tautan, dan banyak lagi. Dengan HTML, Anda dapat:

1. **Mengorganisasi Konten:** Anda dapat mengatur teks, gambar, video, dan elemen lainnya menjadi tampilan yang terstruktur.

2. **Menghubungkan Halaman:** HTML memungkinkan Anda untuk membuat tautan antara halaman web, sehingga pengguna dapat berpindah antara berbagai bagian dari situs Anda.

3. **Menyampaikan Informasi:** Anda dapat menyampaikan pesan, informasi, dan data kepada pengunjung situs Anda.

4. **Membenarkan Struktur:** HTML membantu mesin pencari seperti Google untuk memahami struktur konten Anda, yang dapat mempengaruhi peringkat situs Anda dalam hasil pencarian.

### Memulai dengan HTML

Mari mulai dengan dasar-dasar HTML. HTML bekerja dengan menggunakan **tag** yang mendefinisikan elemen-elemen di halaman web Anda. Setiap tag memiliki makna tertentu. Misalnya, tag `<p>` digunakan untuk menandai paragraf teks, sedangkan `<img>` digunakan untuk menyisipkan gambar.

Contoh penggunaan tag dalam HTML:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Contoh Halaman HTML</title>
</head>
<body>
    <h1>Selamat Datang di Website Saya</h1>
    <p>Ini adalah paragraf pertama saya.</p>
    <img src="gambar.jpg" alt="Deskripsi Gambar">
    <a href="https://www.contohwebsite.com">Klik di sini untuk kunjungi situs lain.</a>
</body>
</html>
```

Dalam contoh di atas, kita menggunakan tag seperti `<html>`, `<head>`, `<body>`, `<h1>`, `<p>`, `<img>`, dan `<a>` untuk membuat halaman web sederhana.
